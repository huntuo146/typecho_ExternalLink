ExternalLinkGuard 是一款专为 Typecho 博客系统开发的外链安全增强插件。当访客点击站外链接时，插件会拦截该请求并引导至中转提示页面，提醒用户注意外部站点风险及个人信息安全。
功能特性
 * 自动识别逻辑：精准识别并拦截非本站域名的所有外部链接。
 * 统一中转页面：提供标准化的外链中转提示页，提升站点专业感。
 * 白名单管理：支持自定义域名白名单，信任域名（如搜索引擎、社交媒体）可直接跳转。
 * 自定义文案：支持在后台灵活修改安全提示文本，适应不同站点的语言风格。
 * 高兼容性：不修改 Typecho 核心文件及主题源码，兼容主流 Typecho 主题。
目录结构
ExternalLinkGuard/
├── Plugin.php          # 插件核心：负责配置注册、钩子绑定及逻辑处理
├── Action.php          # 插件动作：负责外链中转页面的渲染与跳转逻辑
└── README.md           # 插件使用说明文档

安装方法
 * 下载插件：下载或克隆本插件源代码。
 * 重命名文件夹：确保插件目录名称为 ExternalLinkGuard。
 * 上传路径：将文件夹上传至 Typecho 安装目录下的 usr/plugins/ 目录。
 * 激活插件：登录 Typecho 后台，进入“插件管理”界面，点击“激活”按钮。
插件配置说明
激活插件后，点击“设置”进入配置页面：
| 配置项 | 功能描述 | 输入示例 |
|---|---|---|
| 外链白名单 | 设置不触发中转提示的域名，支持子域名匹配 | google.com (每行一个) |
| 提示文案 | 设置显示在中转页面的核心警告或提示信息 | 您即将离开本站，请注意保护隐私 |
> 说明：白名单功能会自动匹配子域名。例如配置 github.com 后，help.github.com 也将被自动放行。
> 
工作流程
 * 链接检测：插件在页面加载时自动扫描链接。
 * 域名校验：判断链接域名是否属于本站或处于白名单列表中。
 * 重定向处理：若非白名单外链，链接将被重定向至插件内置的 Action 路径。
 * 安全提示：展示中转页面，列出目标 URL 及安全警告。
 * 用户确认：用户主动点击“继续访问”后，完成向目标站点的跳转。
兼容性与安全
 * 系统要求：适用于 Typecho 1.2 及以上版本。
 * 隐私保护：插件仅进行逻辑跳转，不记录访客点击日志，不收集任何用户隐私数据。
 * 轻量化：不依赖任何第三方 JavaScript 库或外部 API，运行效率高。
