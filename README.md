🛡️ ExternalLinkGuard
> 为您的 Typecho 博客构建一道坚实的外链安全防线。
> 
ExternalLinkGuard 是一款专为 Typecho 设计的安全增强插件。当访客点击站外链接时，插件会无感拦截并引导至中转提示页面，提醒用户注意外部站点风险，有效提升站点的专业形象与用户信任度。

✨ 功能亮点
 * 🧠 智能识别引擎：自动分析页面链接，精准区分站内与站外地址。
 * 🌉 安全中转页面：提供统一、美观的过渡提示页，降低跳出风险。
 * 🏳️ 灵活白名单：支持域名白名单配置，常用信任站点（如 GitHub, Google）一键放行。
 * ✍️ 高度自定义：支持在后台自由修改提示文案，匹配站点装修风格。
 * 🧩 零侵入设计：无需修改主题源码，即插即用，兼容主流 Typecho 主题。
📂 目录结构
ExternalLinkGuard/
├── Plugin.php          # 核心引擎：处理配置、注册钩子与拦截逻辑
├── Action.php          # 中转终端：负责渲染安全提示页面并执行跳转
└── README.md           # 项目说明书

💿 安装指南
只需简单四步即可完成部署：
 * 📥 下载：克隆或下载本插件的最新版本压缩包。
 * 🏷️ 重命名：解压后，务必将文件夹命名为 ExternalLinkGuard。
 * ⬆️ 上传：将该文件夹上传至网站目录 usr/plugins/ 下。
 * ✅ 激活：登录 Typecho 后台 → 「插件管理」，找到本插件并点击 激活。
⚙️ 配置详解
激活插件后，点击插件旁的 「设置」 按钮进入配置面板：
| 配置面板项 | 功能描述 | 填写示例 |
|---|---|---|
| 🏰 外链白名单 | 设置信任域名，白名单内的链接将直接跳转，不显示提示页。 | github.com
google.com
(一行一个) |
| 💬 提示文案 | 显示在中转页面上的核心安全警告信息，支持 HTML。 | 您即将离开本站，请注意账号财产安全。 |
> 💡 TIPS：白名单功能智能支持子域名泛解析。例如填写 baidu.com，则 tieba.baidu.com、wenku.baidu.com 等都会被自动放行。
> 
🔄 工作原理
用户的每一次点击都会经过以下安全流转：
👉 用户点击链接
↓
🔍 插件实时检测 (判断是否为站外域名)
↓
⚖️ 白名单校验 (是否在信任列表中？)
↓
├── [是] → 🟢 直接跳转目标网站
│
└── [否] → 🛑 拦截请求
↓
🛡️ 展示安全中转页 (显示自定义提示文案)
↓
🌐 用户确认后前往
🛡️ 安全承诺与兼容性
 * 运行环境：完美支持 Typecho 1.2.0 及以上版本。
 * 隐私保护：本插件坚持 「三不原则」 —— 不收集用户数据、不记录点击日志、不追踪用户行为。一切逻辑均在服务器本地完成。
 * 纯净轻量：不依赖 jQuery 等任何第三方库，不拖累站点加载速度。
